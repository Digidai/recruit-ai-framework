Recruitment & AI Hiring Framework

一个可维护的「招聘 × AI 招聘」工具地图（数据驱动），灵感来自 OSINT Framework。

工具树
  • 点击「文件夹」节点展开/收起。
  • 点击「链接」节点会在新标签页打开。
  • 点击带 M 的节点会弹窗让你输入关键词，并把关键词替换进 URL 里的 {query}。

    如何扩展
    1. 编辑 docs/tarf.json（添加/修改节点）。
    2. 本地预览：在项目根目录运行 python -m http.server 8000，然后打开 http://localhost:8000/docs/。
    3. 发布：把 docs/ 作为静态站点部署（GitHub Pages / Cloudflare Pages / Vercel 等）。

    ⚠️ 仅用于招聘工具导航与学习研究；使用任何自动化/AI 招聘能力时请遵守适用法律法规与隐私要求，并进行必要的偏差评估与人类复核。
